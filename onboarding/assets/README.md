# Onboarding Dashboard Tile Images

Place custom images here for each dashboard tile. If an image is missing, a colored placeholder is shown.

- `monitoring.png` - Monitoring application tile
- `step7_exchange.png` - Step7 Exchange blocks generation tile
- `st_block.png` - S&T block generation tile

Recommended size: 120x120 pixels or larger (will be scaled).
